# 5-Gram Language Model (n = 5)

## Author Style
Arthur Conan Doyle

## Dataset Source
Project Gutenberg

## Book Used
The Adventures of Sherlock Holmes

## Objective
To implement a 5-gram statistical language model that generates text in the writing style of Arthur Conan Doyle.

## Methodology
- Text preprocessing (lowercasing, punctuation removal)
- Tokenization
- Construction of 5-grams
- Maximum Likelihood Estimation
- Random sampling for text generation

## How to Run
```bash
python ngram.py
